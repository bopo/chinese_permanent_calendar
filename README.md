# 简介

Chinese permanent calendar

这是一个通过python查询中国万年历相关信息的的包，可以根据阳历日期或阴历日期查询，查询范围从1970-01-01到2099-12-31，返回结果包括：

- 公历信息：节日、星座
- 农历信息：宜、忌、冲、煞、神位、胎神、周易、五行、天干地支、彭祖百忌、董公择日法
